You're deep in the zone on a new project, your terminal alive with a flurry of commands. All is well. Some time passes—a day, a week—and you return, ready to pick up where you left off. But... what was that command again? The one with the obscure flag, the long path, and the precise parameter that took longer to find in your history than it did to execute.

History Book is the solution to this all-too-common problem. It acts as a curated logbook for your project's most vital commands. At the end of a session, simply run history_book add. It intelligently scrapes your recent terminal activity, allowing you to select and save the commands that matter. Enrich them with descriptions and tags for context. The next time you return to the project, your personalized command arsenal is ready and waiting, allowing you to list, review, and execute them instantly.

Organize Your Shell History, Master Your Workflow

History Book is a powerful command-line tool designed to help developers and power users effortlessly manage their frequently used shell commands for specific projects. Say goodbye to cluttered global history and endless searching – History Book keeps your essential commands organized, searchable, and ready to run.

Key Features

Effortless Command Scraping

Pull recent commands directly from your bash, zsh, or fish shell history with an intuitive text-based interface. Select exactly what you need, and History Book handles the rest.

Intelligent Organization

Assign short names, detailed descriptions, and comma-separated tags to your commands. This makes finding and understanding complex commands a breeze, even months later.

Quick Execution & Control

Execute saved commands by their short name directly from the CLI. Control output verbosity with per-command or global quiet flags, ensuring your terminal stays clean.

Interactive Editing

Modify any property of your saved commands – name, description, tags, or quiet status – through a user-friendly interactive editing interface.

Get Started in Minutes

Installing History Book is straightforward. Just clone the repository and run the installer script: 

git clone https://github.com/your-username/history-book.git
cd history-book
./install.sh

The installer will set up a virtual environment, install dependencies, and create a `history_book` launcher in `~/.local/bin`. You may need to add `~/.local/bin` to your `PATH` if it's not already configured.

Simple Usage

Add Commands:

history_book add

Interactively select and save commands from your shell history.

List Commands:

history_book list
history_book list --tags "build,docker"

View all saved commands or filter by tags.

Run Commands:

history_book run my_dev_setup
history_book run deploy_prod --quiet

Execute a command by its short name, with optional quiet mode.

Edit Commands:

history_book edit

Interactively modify details of your saved commands.
